# Minha Carteira

## Sobre

Minha Carteira é um Dashboard de controle de gastos.

---

## Tecnologias Utilizadas

- [React.js]()
- [Typescript]()
- [Styled-components]()

---

## Como baixar

```bash

        # Clone o Repositório
        $ git clone https://github.com/eugenio-silva/Minha-Carteira.git

        # Entre no diretório
        $ cd Minha-carteira

        # Instale as dependências
        $ yarn install

        # Inicie a aplicação
        $ yarn start
```
